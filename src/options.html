Button

Which button will you hold down to drag?
WARNING: The middle button may not work on Windows due to a bug in the browser, which will be fixed in Chrome 5.

Keys

Which extra keys will you hold down?
Tip: any unchecked keys will disable dragging when held down


Try unchecking this if you notice any delays

Grab and drag

Grab-and-drag style scrolling will be enabled instead of the scrollbar anywhere style.

Pointer Locking

Double-click to lock pointer and scroll with free mouse movement.
Tip: exit pointer locking by clicking again or pressing escape

Scaling

Mouse motion will be magnified by this factor. A negative number will invert scrolling. 100 is default. 		% of original scroll speed

Top Speed

The maximum speed at which the page will glide after you release it. Enter 0 to disable gliding. 		pixels per second

Friction

How quickly the page will come to a stop when gliding. 		velocities per second

Blacklist

Disable this extension on the configured domains and all of their subdomains.
One domain name per line.
